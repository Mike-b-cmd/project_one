# project_one
# Исследование возможностей чат-ботов с ИИ в добывающей промышленности

## Автор: Михаил (Студент ПСУ (НПИ) / Цифровая кафедра)

## Цель исследования
Сравнить качество решения прикладных задач горнорудной отрасли с помощью зарубежного (DeepSeek) и отечественного (GigaChat) ИИ-ассистентов в рамках стажировки в ООО «ИНТЕРНЕТ-ФРЕГАТ».

## Используемые модели
1. **Зарубежный бот:** DeepSeek-V3 / R1 (ориентирован на глубокую техническую логику).
2. **Отечественный бот:** GigaChat Pro (Сбер) (ориентирован на российскую нормативную базу и ГОСТы).

## Выбранные кейсы
1. **Кейс 1: Диагностика неисправностей насосного оборудования.**
   * *Задача:* Анализ ситуации с падением давления масла, ростом температуры и вибрации.
2. **Кейс 2: Анализ нормативно-технической документации (Охрана труда).**
   * *Задача:* Извлечение требований к СИЗ из регламентов по взрывным работам.

## Методология
1. Формирование идентичных промптов на русском языке.
2. Тестирование моделей в режиме "Zero-shot".
3. Сравнение ответов по критериям: техническая точность, знание локальных норм, структурированность.

## Основные выводы
* **DeepSeek** блестяще справился с физикой процессов (описал кавитацию и износ подшипников), но давал общие международные рекомендации.
* **GigaChat** показал превосходство в знании российских требований ПБ и охраны труда, корректно используя специфическую терминологию (СИЗ, наряды-допуски).
* Интеграция обоих типов агентов в системы типа «Фарватер» позволит значительно сократить время принятия решений на производстве.
